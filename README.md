**Formula**: `LPI = (Bio_Updates / Enrol) Ã— (Demo_Updates / Enrol)`

**Formula**: `UCP = P(Bio|Demo) Ã— P(Demo|Enrol)`
**Finding**: 10% improvement in Step 1 â†’ +33% final completion.

#### **3. Migration Directionality Index (MDI)**
