> [!TIP]
> **Why This Matters**: Enrollment patterns (birth cohorts) get lost when merged with migration data. Domain separation revealed the missing college cohort that merged analysis masks.

### **3 Novel Metrics Created**

#### **1. Lifecycle Progression Index (LPI)**

**Formula**: `LPI = (Bio_Updates / Enrol) √ó (Demo_Updates / Enrol)`

**Purpose**: Measures ecosystem health - what % of citizens complete the full journey?

**Finding**: National LPI = 0.08 ‚Üí 92% dormancy crisis

**Innovation**: UIDAI tracks individual steps, but nobody measures **completion rate** - we created the metric

---

#### **2. Update Cascade Probability (UCP)**

**Formula**: `UCP = P(Bio|Demo) √ó P(Demo|Enrol)`

**Purpose**: Identifies the highest-ROI policy lever via conditional probability

**Finding**: 10% improvement in Step 1 ‚Üí +33% final completion (exponential effect)

**Innovation**: Quantifies **cascading effects** - proves early engagement has 3x ROI vs late-stage interventions

---

#### **3. Migration Directionality Index (MDI)**

**Formula**: `MDI = (Out_Migration - In_Migration) / (Out + In)`

**Purpose**: Classifies districts as emigration sources vs immigration destinations

**Finding**: 
- **+1.0 MDI** = Pure emigration (e.g., Khairthal-Tijara) ‚Üí Deploy retention programs
- **-1.0 MDI** = Pure immigration (e.g., Bengaluru Urban) ‚Üí Scale capacity

**Innovation**: Enables **targeted strategies** instead of one-size-fits-all deployment
